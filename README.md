Contact Management System in Java

A simple console-based Contact Management System written in Java that allows users to add, view, search, and delete contacts. This project is perfect for beginners learning Java, ArrayLists, and basic OOP concepts.

Features

Add a new contact with name, phone number, and email.

View all saved contacts in a structured format.

Search for a contact by name.

Delete a contact by name.

Menu-driven console application for easy interaction.

Requirements

Java JDK 8 or higher

Any Java IDE (Eclipse, IntelliJ, VS Code) or terminal with javac and java installed
